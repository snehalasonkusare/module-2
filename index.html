Lesson 01 — Foundations

HTML Basics

HTML (HyperText Markup Language) is the skeleton of every webpage. It uses tags to wrap content and give it meaning. Edit the live playground on the right to see your changes instantly.

01

Document Structure

Every HTML file starts with a doctype declaration and wraps content in <html>, <head>, and <body> tags.

html 

<!-- Basic skeleton -->
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <!-- content here -->
  </body>
</html>

02

Headings & Paragraphs

Use <h1>–<h6> for headings (h1 = largest) and <p> for body text.

html 

<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<p>A paragraph of text.</p>

03

Links & Images

The <a> tag creates hyperlinks using the href attribute. The <img> tag embeds images with src and alt.

html 

<a href="https://example.com">
  Visit Example
</a>

<img src="photo.jpg"
     alt="Description">
